#  esms-工资管理系统
## 2018年7月-工资管理系统-东软班培训

## 开发环境
    1）数据库使用MySQL5.7
    2）后台使用SSM框架
    3）前端UI框架使用Layui
    4）服务器：Tomcat8.0.35

## 系统功能
#### 系统拥有员工端和管理员端
##### 员工端：
    1）个人信息管理模块
    2）考勤查询模块
    3）工资查询模块
##### 管理员端：
    1）部门管理模
    2）岗位管理模块
    3）员工信息模块
    4）职称奖金管理
    5）工龄奖金管理
    6）工资项管理
    7）月考勤管理
    8）工资结算管理
    9）数据图表管理
    10）文件上传下载管理
